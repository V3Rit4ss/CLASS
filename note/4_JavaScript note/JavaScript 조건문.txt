5. switch 조건문
 switch(비교할 값) {
case 값: 
	문장; break;
case 값: 
	문장; break;
Default:
	문장; break;
}

6. 삼항연산자
 (불 표현식)? (참일 때 실행하는 문장) : (거짓일 때 실행하는 문장);

자바와 매우 흡사하기때문에 자바에서 알고있는 if문, if ~ else , 중첩 if , switch, 삼항 연산자 를 쓰면 된다.
허나 조금은 다른 짧은 조건문은 살짝 다르니 알아두자.

7. 짧은 조건문
 짧은 조건문은 논리연산자의 특성을 조건문으로 사용한다. 다음 논리합 연산자를 사용한 표현식은 뒤에 어떠한 값이 들어가도 항상 참이므로 우변을 실행하지 않는다.

 <script>
        var su = Number(prompt('자연수를 입력하세요'));
      //  true || '이 부분 수행 안함';
     //   false || '이 부분 수행';
        //   false && '이 부분 수행 안함';
        (su % 2 == 0) || alert('홀수입니다.');
        (su % 2 == 0) && alert('짝수입니다.');
    </script>
