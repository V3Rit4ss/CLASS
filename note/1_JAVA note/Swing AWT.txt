스윙(Swing)

3.  Swing 컴포넌트?
Swing은 AWT보다 진보된 기능이라고 생각하면 된다. 진보됐다는 것은 AWT보다 나중에 만들어져, 복잡하고 화려한 컴포넌트를 제공한다.

- 모든 AWT 기능 + 추가된 풍부하고 화려한 고급 컴포넌트.
- AWT 컴포넌트를 모두 스윙으로 재작성 가능, AWT 컴포넌트 이름 앞에 J자를 붙임.

AWT 컨테이너 : Panel , Frame, Applet, Dialog 등등
Swing 컨테이너 : JPanel , JFrame, JApplet, JDialog 등등.


차이점 :
AWT 
-초기에 지원한 GUI
-운영체제가 사용하는 자원을 사용
-플랫폼 의존적
-컴포넌트 용량이 큼.
-컴포넌트 갯수가 적음.


Swing
-상대적으로 늦게 나온 GUI
-컴포넌트가 자바로 되어있음.
-플랫폼 독자적.
-캄포넌트 용량이 가벼움.
-컴포넌트 갯수가 많음.
