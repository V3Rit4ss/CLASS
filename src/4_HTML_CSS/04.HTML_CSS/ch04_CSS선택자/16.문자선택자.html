벌써 올해의 마지막 달이 왔어요!

연말은 그동안 못 만났던 사람들을 만나면서 북적거리며 설레는 달인데 올핸 코로나가 있어 아쉽게도 그러지 못할 거 같아요. 최대한 외출을 자제하고 있지만, 답답함에 한적한 곳으로 안전한 여행을 찾는 분들이 많은데요. 그래서 캠핑 장비에 대한 수요도 전년 대비 2배 이상 늘었습니다. 캠핑은 보통 봄,가을에 많이들 떠난다고 생각하지만 이에 못지않게 겨울 캠핑도 인기가 많답니다! 겨울 캠핑을 아무래도 준비할 것이 조금 더 많은데요, 캠핑을 처음 준비하시는 분들을 위해 겨울 캠핑에 필요한 몇 가지 소개해드립니다. 우선, 가장 중요한 건 추위에 대비하는 일이예요. 전기장판이나 온수매트를 이용해 바닥을 따뜻하게 해주고 따뜻한 잠자리를 위해 침낭을 준비해야합니다. 난로를 사용하신다면 환기는 필수입니다~!!

난방이 모두 준비됐다면 뭐니뭐니 해도 가장 중요한 건 먹는 게 아닐까 싶어요. 보통 캠핑을 할 땐 조리가 복잡한 음식보다는 조리 방법이 간단한 음식들을 선호하고, 양념이 필요한 음식들은 양념까지 모두 준비해가는 게 일반적이에요. 아무래도 집에서 준비하는 것보다 환경이 열악하기 때문에 준비할 수 있는 건 미리 집에서 다 준비하고 캠핑장에선 바로 조리해서 먹는 거죠!
